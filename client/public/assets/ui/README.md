# UI Assets

Place your UI assets here:

- `banner.png` - The banner image that appears above castles
- `gate-icon.png` - The gate icon button for entering castles

## Asset Specifications

### banner.png
- Should be a horizontal banner/scroll design
- Recommended size: 300-400px wide, 80-100px tall
- Should have transparent or matching background
- Pixel art style to match the game

### gate-icon.png
- Gate/castle entrance icon
- Recommended size: 32x32 or 48x48 pixels
- Transparent background
- Pixel art style
