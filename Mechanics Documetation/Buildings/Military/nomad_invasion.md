# Nomad Invasion Event

## Overview
The Nomad Invasion is a recurring NPC event where players fight against nomad camps that appear throughout the Great Empire. During the event, players can earn Khan Tablets by defeating nomad camps, which can be exchanged for unique rewards at the Herald of the Invasion tent outside the main castle.

**Event Start Date**: July 9, 2013  
**Classic Era**: Pre-2018 mechanics (before difficulty modes and legendary features)

## Event Structure

### Event Location
- Takes place in the Great Empire (main kingdom map)
- Nomad camps spread around the main castle
- Herald of the Invasion tent appears outside main castle
- Four nomad camps available for each player

### Event Duration (Original)
- Regular recurring event
- Time-limited availability
- Cooldown periods between events
- Attack cooldowns on individual camps

## Nomad Camps

### Camp Mechanics (Pre-2015)
**Original System**:
- Fixed difficulty camps
- Fixed defensive compositions
- No level progression
- Standard respawn timer
- Consistent challenge across all players

### Camp Mechanics (Post-April 2015 Revision)
**Dynamic System**:
- Camps increase in level when defeated
- Short attack cooldown after each victory
- Cooldown increases with camp level
- Maximum level camps have 5.5 hour cooldown
- Difficulty scales with player level

### Camp Defenders
Nomad camps feature unique defensive units:
- Special nomad soldier types
- Unique combat statistics
- Defensive tools deployed
- Wall, gate, and moat fortifications
- Castellan bonuses

## Combat Mechanics (Classic Era)

### Defensive Tools Distribution (Pre-September 2013)
- Defense tools spread across waves
- Some tools in early waves only
- Wave-specific defensive bonuses

### Defensive Tools Distribution (Post-September 2013)
- Defense tools distributed across ALL waves
- More consistent defensive bonuses
- Harder to nullify defenses
- Requires better strategy for success

### Attack Strategy Fundamentals
1. **Espionage**: Always spy before attacking (100% information)
2. **Tool Selection**: Use appropriate siege tools for camp defenses
3. **Army Composition**: Balance melee and ranged soldiers
4. **Wave Tactics**: Multiple waves may be necessary
5. **Resource Tools**: Use loot tools to maximize tablet gains

## Khan Tablets

### Earning Khan Tablets
- Primary reward for defeating nomad camps
- Amount varies by camp difficulty and player level
- Special tools can increase tablet gains
- Base tablet reward per camp victory

### Tablet Bonuses
Certain tools provide Khan Tablet bonuses:
- **Khan Chest**: +3% Khan Tablet bonus (17 rubies)
- **Rolling Shield**: -20% enemy ranged soldiers + 3% tablet bonus (21 rubies)
- **Fire Cart**: -25% enemy gate protection + 3% tablet bonus (23 rubies)
- **Tree Trunk Ladder**: -25% enemy wall protection + 3% tablet bonus (20 rubies)

### Spending Khan Tablets
Exchange at Herald of the Invasion tent for:
- Unique soldiers
- Special tools
- Decorations
- Resources
- Build items
- Equipment (later additions)

## Strategic Approach (Classic Era)

1. **Build Event Tools**: Stock up on Khan Chests and specialized tools
2. **Army Ready**: Ensure sufficient troops available
3. **Tool Production**: Have siege tools ready (especially mantlets)
4. **Resource Management**: Maintain tool production capacity

### Attack Execution
1. **Spy Mission**: Send spies for 100% intelligence
2. **Analyze Defense**: Identify defensive composition
3. **Select Tools**: Choose appropriate siege and event tools
4. **Army Composition**: Balance offense for defender types

### Post-Attack Management
1. **Tool Restocking**: Immediately restart mantlet production
2. **Casualty Recovery**: Recruit replacement troops
3. **Tablet Management**: Decide on spending vs. saving
4. **Camp Monitoring**: Track cooldown timers for next attack

## Tool Consumption

### High-Consumption Tools
**Mantlets** (Critical Resource):
- Essential for reducing ranged defender casualties
- Very high consumption rate
- Approximately 720 mantlets per 10 attacks
- Must maintain continuous production
- Highest priority tool for event success

**Other Siege Tools**:
- Ladders and rams consumed per attack
- Less critical than mantlets
- Standard siege workshop tools work fine

## Rewards Optimization

### Maximizing Khan Tablets
1. **Use Khan Chests**: 3% bonus adds up significantly
2. **Multiple Tools**: Stack bonuses (Fire Cart + Khan Chest)
3. **Efficient Attacks**: Minimize failed attacks
4. **Camp Progression**: Higher level camps give more tablets
5. **Continuous Activity**: Attack consistently during event

### Resource Benefits
1. **Piggy Banks**: Increase coin gains from victories
2. **Experience Bonuses**: Level up faster during event
3. **Loot Tools**: Maximize resource gains from camp attacks

## Event Changes Timeline (Pre-2018)

### Initial Release (July 2013)
- First implementation of nomad camps
- Basic reward system
- Fixed camp difficulties

### Early Adjustments (2013-2014)
- **September 2013**: Reduced castellan bonuses and troop numbers in higher camps
- **September 2013**: Defense tools now distributed across all waves
- **October 2014**: Reduced event duration, camp cooldowns, and lootable tablets

### Major Revision (April 2015)
- **New Appearance**: All nomad camps received visual updates
- **Dynamic Difficulty**: Camps now level up when defeated
- **Cooldown System**: Short cooldowns between attacks, longer for max level
- **Level Scaling**: Camp difficulty adjusts to player level
- **Updated Rewards**: Revised shop offerings

### Continued Development (2015-2017)
- **June 2015**: Added alliance contest variation
- **June 2015**: Adjusted camp strength for levels 20-69
- **February 2016**: Shop updates and reduced lootable coins
- **April 2016**: Build items added to shop
- **August 2016**: Activity rewards system implemented
- **July 2017**: Khan's Revenge added (advanced feature)
- **September 2017**: Equipment sets added as rewards

## Tips for Success (Classic Era)

### Essential Strategies
1. **Mantlet Priority**: Keep workshop producing mantlets constantly
2. **Spy Always**: Never attack without intelligence
3. **Tool Preparation**: Stock event tools before attacking
4. **Army Size**: Bring sufficient troops to minimize casualties
5. **Timing**: Plan attacks around camp cooldowns

### Resource Management
1. **Tool Production**: Start mantlet production early
2. **Troop Reserve**: Maintain surplus troops for continuous attacks
3. **Ruby Conservation**: Save rubies for critical situations
4. **Tablet Spending**: Plan purchases based on priorities

### Efficiency Tips
1. **Attack Rotation**: Hit different camps during cooldowns
2. **Production Timing**: Align tool production with attack schedule
3. **Alliance Coordination**: Share strategies with alliance
4. **Learning Curve**: Each event improves understanding

### Common Mistakes to Avoid
1. **No Mantlets**: Attacking without adequate mantlets
2. **Weak Spy**: Attacking with insufficient intelligence
3. **Overconfidence**: Sending too few troops
4. **Tool Neglect**: Not using event-specific bonus tools
5. **Poor Planning**: Not managing cooldown timers

## Classic Era Significance

The Nomad Invasion event was significant in classic Good Game Empire for several reasons:

1. **PvE Content**: Provided structured PvE challenges
2. **Unique Rewards**: Offered soldiers and items unavailable elsewhere
3. **Community Building**: Encouraged alliance cooperation
4. **Skill Development**: Taught players advanced combat tactics
5. **Recurring Engagement**: Regular event kept players returning
6. **Competitive Element**: Rankings and rewards drove participation
7. **Resource Sink**: Created demand for tools and troops
8. **Progression Path**: Offered alternative advancement through event rewards

The pre-2018 Nomad Invasion established many core mechanics that shaped player engagement and alliance dynamics in the classic era of Good Game Empire.
